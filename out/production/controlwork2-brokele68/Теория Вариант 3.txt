1. Что такоое абстрактный класс в Java?

Это класс в котором есть хотя бы один абстрактный метод. Этот класс работает только через его наследников, которые должны реализовывать все абстрактные методы.

2. В чем отличие интерфейса от абстрактного класса?

Интерфейс может содержать только абстрактные методы, также методы по умолчанию. Класс может наследоваться только от одного класса, но может реализовывать несколько интерфейсов.

3. Каким образом в Java реализуется композиция?

Когда объекты одного класса являются членами другого класса.

4. Что такое статический и динамический полиморфизм в Java?

Статический - возможность вызывать один и тот же метод с разными параметрами или типами данных. Динамический - возможность вызывать методы наследников через ссылку на объект класса родителя.

5. Каково значение ключевого слово super в Java?

для обращения к методам и полям родительского класса из наследника. Просто super() вызывает конструктор класса родителя, а super.Method() метод класса родителя.
